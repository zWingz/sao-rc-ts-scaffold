# sao-rc-component-ts

> template to create a react component with ts

- code by `typescript`
- use `jest` for unit test
- build doc by `docz`
- bundled by `rollup`
- use `poi` to develop

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao rc-component-ts my-project
```

### From git

```bash
sao zwingz/sao-rc-component-ts my-project
```
